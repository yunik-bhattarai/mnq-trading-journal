# 📈 MNQ Trading Journal

A Python-based trading journal designed for MNQ (Micro E-mini Nasdaq-100 Futures) traders.

## Features

- Add Trades
- Edit Trades
- Delete Trades
- Search Trades
- Trading Statistics
- Daily Summary
- Equity Curve
- JSON Storage
- CSV Export

## Technologies Used

- Python
- JSON
- CSV
- File Handling
- Dictionaries
- Lists

## Future Improvements

- Streamlit Dashboard
- Data Visualization
- Performance Analytics
- Trade Screenshot Upload
